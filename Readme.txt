This is the program to extract different channels from an image.

path => Location of the folder where files are stored. 
stro2 => Location of the Folder where original files read will be stored
strr2 => Location of the Folder where RED channel files will be stored
strg2 =>Location of the Folder where GREEN channel files will be stored
strb2 =>Location of the Folder where BLUE channel files will be stored

You can also use different folders for storing different output channels. In this case all the output files will be stored in the OUTPUT folder.
